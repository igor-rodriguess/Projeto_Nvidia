# RAPIDS, cuDF e cuML

Fontes oficiais:

- NVIDIA RAPIDS: https://rapids.ai/
- cuDF: https://docs.rapids.ai/api/cudf/stable/
- cuML: https://docs.rapids.ai/api/cuml/stable/

RAPIDS acelera ciencia de dados e machine learning classico em GPU. cuDF oferece manipulacao de dados em GPU com API familiar para usuarios de pandas. cuML oferece algoritmos de machine learning acelerados.

Quando recomendar:

- Startup trabalha com dados tabulares, features, scoring, risco, fraude, precificacao ou recomendacao.
- Pipelines em pandas ou scikit-learn estao lentos.
- O ganho esperado vem de acelerar ETL, treinamento, inferencia tradicional ou experimentacao.

Sinais em uma startup:

- Menciona grandes volumes de dados.
- Produto depende de analytics, modelos preditivos ou segmentacao.
- Problema central e throughput de dados, nao necessariamente IA generativa.
