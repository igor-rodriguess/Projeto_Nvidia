# NVIDIA API Catalog

Fonte oficial: https://build.nvidia.com/

O NVIDIA API Catalog permite explorar, testar e usar APIs de modelos e microservicos NVIDIA. Ele e relevante para prototipacao, avaliacao de modelos e integracao rapida de capacidades de IA.

Quando recomendar:

- Startup quer testar modelos antes de operar infraestrutura propria.
- Time precisa prototipar rapidamente assistentes, agentes, busca semantica, visao, fala ou geracao.
- Produto precisa comparar modelos e capacidades antes de definir arquitetura final.

Sinais em uma startup:

- Ainda esta validando qual modelo usar.
- Precisa demonstrar MVP com IA generativa ou multimodal.
- Quer reduzir tempo de integracao inicial.
