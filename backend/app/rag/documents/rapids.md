# NVIDIA RAPIDS

Fonte oficial: https://docs.nvidia.com/rapids/index.html

NVIDIA RAPIDS e um conjunto de bibliotecas para executar pipelines de ciencia de dados e analytics em GPU, mantendo uma experiencia familiar para usuarios Python.

Quando recomendar:

- Startup trabalha com grandes volumes de dados tabulares, features, analytics ou machine learning classico.
- Ha gargalos em pandas, scikit-learn, processamento de dados ou preparacao de features.
- O ganho esperado vem de acelerar ETL, analise exploratoria, treinamento ou inferencia tradicional.

Sinais em uma startup:

- Produto depende de dados, analise, scoring, risco, recomendacao, fraude, precificacao ou otimizacao.
- Equipe usa pipelines Python intensivos em processamento.
